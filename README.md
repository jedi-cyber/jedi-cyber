### 📱 TecnicApp - Sistema RMA
**Aplicación profesional de gestión técnica para talleres de servicio**

🔗 [Ver repositorio](https://github.com/jedi-cyber/TecnicApp)

#### 💡 Características:
- 📋 Gestión integral de tickets y órdenes de servicio
- 🎯 Sistema de prioridades con estados visuales
- ☁️ Modo offline con sincronización automática
- 👥 Colaboración en equipo en tiempo real
- 📊 Dashboard con reportes y estadísticas
- 📄 Exportación a PDF y Excel
- 🔐 Seguridad con EncryptedSharedPreferences
- 🧪 Testing y validación robusta
